# HistorIA 3.0 — Static (GitHub Pages)

Pacote 100% estático (HTML/CSS/JS).

## Publicação (somente GitHub)
Settings → Pages → Deploy from a branch → main / (root)

## Uso
1. Abra `index.html` (ou a URL do GitHub Pages).
2. Termos → salve a **Licença de Uso (Gemini)**.
3. Controles → selecione o modelo (padrão: gemini-2.5-flash).
4. Crie a história.

## Botão Atualizar (correção automática)
Quando um trecho ficar incompleto (ex.: finalização 100% ou início do próximo capítulo),
o botão **Atualizar** tenta **continuar a geração** sem permitir alterações do leitor.
